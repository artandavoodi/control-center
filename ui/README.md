# UI

This directory holds the control and operational surfaces for Control Center.

## Purpose

UI is the visible control layer for the system.

It should include:

- dashboard
- editor
- settings
- registry controls
- future system panels

## Current Reality

The UI layer is minimal at present and serves as the start of the control surface structure.

## Role

UI is the operator-facing layer, not the system brain.

Control Center governs. UI displays and lets operators act.
