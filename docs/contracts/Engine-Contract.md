# Engine Contract

## DSC
Owns tokens, themes, visual rules, and platform adapters.

## Registry
Owns IDs, asset lookup, and object identity.

## UVS
Owns voice identity, tone, and interaction behavior.

## UAPIS
Owns internal communication and interface contracts.

## UI
Owns the control surfaces and operator panels.

## Control Center
Owns orchestration, validation, and system coherence.
