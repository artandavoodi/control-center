# Engine Boundaries

## DSC
Design System Core.
Owns tokens, themes, visual rules, and platform adapters.

## UVS
Unified Voice System.
Owns tone, voice identity, and interaction style.

## UAPIS
Unified API System.
Owns internal message flow, interface contracts, and system communication.

## Control Center
Top-level orchestration and governance layer.
Owns coherence, validation, and cross-system coordination.

## Registry
Owns IDs, object identity, asset lookup, and reference stability.

## UI
Owns control surfaces, dashboards, editors, and settings panels.
