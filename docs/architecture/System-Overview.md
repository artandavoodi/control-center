# System Overview

Control Center is the top-level orchestration and governance layer for the Neuroartan system.

## Core Purpose

The system exists to unify:
- DSC — Design System Core
- UVS — Unified Voice System
- UAPIS — Unified API System
- Registry — identity and asset authority
- UI — control and operational surfaces

## Architectural Truth

- DSC already exists as a distributed design system.
- Control Center is the layer above DSC.
- Control Center is not a rename of DSC.
- Control Center governs coherence, validation, and coordination.

## Key Gaps

- runtime enforcement
- full registry graph
- formal interface contracts
- closed-loop validation
- synchronized cross-platform propagation

## Working Doctrine

- scan first
- verify ownership first
- root fix only
- no overlays
- no workaround layers
- no hardcoded drift
- terminal output is source of truth
