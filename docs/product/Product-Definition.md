# Product Definition

Neuroartan is evolving into a studio-led ecosystem where AI and human control work together.

## Product Goal

Build a system where users can:
- design visually
- control style through tokens and themes
- assign IDs to assets
- manage components
- create templates
- use a marketplace inside the ecosystem
- develop public or private projects with one unified foundation

## Design Philosophy

- human taste leads
- AI structures and propagates
- registry gives identity
- runtime enforces coherence
- studio turns style into a system
